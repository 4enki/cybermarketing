Исходники и структура папок и файлов

Исходный код статичной версии лежит на ГитХабе: @@git. Документ со списком планов, багов и правок размещён в гугл-документе.

Техническую информация по развёртованию сборки в локальном варианте можно найти в файле README.md. Готовые к употреблению шаблоны расположены в дирректории dist/ (сгенерирована автоматически).

Показать стуктуру папок и файлов
├── desktop/                 # Исходники для страндартного вида
│   ├── images/              # Изображения и графика
│   │   ├── i/               # Favicons
│   │   └── svg/             # SVG иконки
│   ├── resources            # Статические файлы для копирования в /dist
│   ├── scripts/             # Скрипты
│   ├── styles/              # Стили
│   │   ├── _debug/          # Стили для отладки
│   │   ├── base/            # Стандратные стили
│   │   ├── helpers/         # Помощники
│   │   └── style.scss       # Главный стилевой файл
│   └── templates/           # Разметка
│       ├── blocks/          # Блоки
│       └── pages/           # Страницы
├── mobile/                  # Исходники для мобильного вида
│   ├── scripts/             # Скрипты мобильной версии
│   ├── styles/              # Стили мобильной версии
│   └── templates/           # Разметка мобильной версии
├── gulpfile.js/             # Конфиг Gulp.js
│   ├── tasks/               # Отдельные файлы с задачами
│   ├── paths.js             # Пути к ресурсам проекта
│   └── index.js             # Основные задачи
├── dist/                    # Сборка проекта (автогенерация)
│   ├── mobile/              # Разметка мобильной версии
│   ├── desktop/             # Разметка стандратной версии
│   ├── assets/              # Подключаемые ресурсы
│   │   ├── mobile/          # Ресурсы для мобильной версии
│   │   ├── desktop/         # Ресурсы для стандратной версии
│   │   └── images/          # Общие изображения
│   ├── ui.html              # UI: Контролы и компоненты
│   └── index.html           # Карта сайта с прогрессом работ
├── .editorconfig            # Конфигурационный файл IDE
├── .gitignore               # Список исключённых файлов из Git
├── browserslist             # Список поддерживаемых браузеров для Автопрефиксера
├── package.json             # Список модулей и прочей информации для NPM
├── yarn.lock                # Список модулей и прочей информации для Yarn
├── sass-lint.yml            # Конфиг для линтинга scss
└── README.md                # Документация проекта

Подробная техническая документация размещена в проекте SPT, ГитХаб.
